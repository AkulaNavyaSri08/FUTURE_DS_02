# FUTURE_DS_02
# Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analyzing customer churn and retention patterns using Power BI. The dashboard helps identify customers who are likely to leave, understand factors influencing churn, and provide insights to improve customer retention.

## Dataset
- Customer Churn Dataset
- 1200+ customer records
- Includes customer demographics, services, contract type, tenure, charges, and churn status

## Tools Used
- Power BI
- CSV Dataset
- Data Analysis

## Dashboard Features
- Total Customers KPI
- Churned Customers KPI
- Retained Customers KPI
- Churn Rate Analysis
- Churn by Contract Type
- Churn by Gender
- Average Monthly Charges by Churn
- Average Tenure by Churn
- Interactive Filters (Slicers)

## Key Insights
- Month-to-month contract customers have the highest churn rate.
- Customers with shorter tenure are more likely to leave.
- Long-term contracts improve customer retention.
- Monthly charges influence customer churn behavior.

## Recommendations
- Encourage customers to switch to long-term contracts.
- Provide loyalty rewards for existing customers.
- Improve customer engagement during the initial months.
- Monitor high-risk customers and offer retention incentives.
